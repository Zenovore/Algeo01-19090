## Cara menjalankan
### Alternatif 1
1. `make compile` untuk compile source
1. jalankan kode yang ada di `bin/tubes` menggunakan `cd bin && java
   tubes.Driver`
### Alternatif 2
1. `make run` untuk menjalankan langsung
2. Gunakan program
### Alternatif 3
1. `make run jar` untuk menggabungkan menjadi 1 jar dan menjalankan\
2. Gunakan program
